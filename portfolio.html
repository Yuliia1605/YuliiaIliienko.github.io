My projects

  • LinguaSaur

    It is a free website where you can read adapted English books. The texts are simplified for different levels—from beginner to advanced—and include built-in dictionaries, audio, and word explanations right in the text. It is easy to read, even if you do not know much English yet. LinguaSaur

  • Khan Academy - Math for Engineering

    It is a free website where you can learn the math engineers use: algebra, trigonometry, calculus, differential equations, and more. Everything is explained simply, with step-by-step examples—even if you are not very good at math yet. Khan Academy

  • Hedenstedbib.dk

    It is a free online library offering access to thousands of classic books whose copyrights have expired. All texts can be read online or downloaded in PDF, EPUB, or plain text formats. Hedenstedbib

  • freeCodeCamp

    It is a free, open-source platform that offers full programming courses, including Java. It provides video lessons, articles, and hands-on projects—all completely free. freeCodeCamp

  • The Odin Project

    This website focuses mainly on web development but also includes helpful links and guidance for learning Java, especially for those who want to understand programming fundamentals before diving into web technologies. The Odin Project
